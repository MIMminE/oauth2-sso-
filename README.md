- sso-vue-app-1 과 sso-vue-app-2 는 vue (TS) 로 작성된 SSO 클라이언트 애플리케이션입니다.
- 각 클라이언트 애플리케이션은 sso-back-app-1 과 sso-back-app-2 로 작성된 SSO 백엔드 애플리케이션과 통신합니다.
- sso-back-app-1 과 sso-back-app-2 는 하나의 중앙 인증 서버 (sso-auth-server) 와 연동되어 사용자 인증을 처리합니다.
- sso auth server 는 OAuth2 / OpenID Connect 프로토콜을 사용하여 인증 및 권한 부여를 관리합니다.
- 인증 관리 서버는 클라이언트 -> 백엔드를 통해 온 요청을 받아 사용자에게 Oauth2 인증 플로우를 수행할 수 있도록 인증 페이지를 제공한다.
- 사용자가 인증을 수행하면 인증 서버에 Oauth2 토큰을 저장하고 이와 연결된 세션 쿠키를 발급한다.
- 이후 클라이언트 애플리케이션은 백엔드 애플리케이션과 통신할 때 세션 쿠키를 사용하여 인증된 요청을 보냅니다.
- 백엔드 애플리케이션은 세션 쿠키를 검증하여 사용자의 인증 상태를 확인하고, 필요한 경우 인증 서버와 통신하여 추가 정보를 가져올 수 있습니다.
- 이 구조를 통해 여러 클라이언트 애플리케이션이 하나의 중앙 인증 서버를 통해 인증을 처리할 수 있습니다.
